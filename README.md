# Electric Bus Tracking System (Database Project)

## Overview

The Electric Bus Tracking System is a database-based application designed to support real-time monitoring and management of electric buses. The system helps passengers track bus locations, view routes, and estimate arrival times.

This project focuses on designing and implementing a structured database that manages transportation data such as buses, drivers, routes, schedules, and tracking information.

## Features

* Real-time bus location tracking
* Bus route and stop management
* Driver duty assignment
* Bus schedule management
* Passenger route search
* Estimated arrival time information

## Technology Stack

* **Database:** MySQL
* **Backend:** Node.js + Express.js
* **Mobile App:** Flutter
* **Real-time Updates:** Socket.io
* **Maps Integration:** Google Maps API
* **Authentication:** JWT + bcrypt
* **Version Control:** Git & GitHub

## User Roles

* **Passenger:** View bus locations, search routes, and check arrival times.
* **Driver:** Start duty, share live location, and view assigned routes.
* **Administrator:** Manage buses, routes, schedules, and drivers.

## Project Purpose

The goal of this project is to design a database system that can efficiently manage transportation data and support future development of a real-time electric bus tracking application.
